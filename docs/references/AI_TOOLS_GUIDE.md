# Critérios de Escolha

Antes de utilizar qualquer ferramenta, responder:

- Qual problema ela resolve?
- Existe alternativa mais simples?
- Possui documentação oficial?
- É utilizada no mercado?
- Possui API?
- Possui comunidade ativa?
- Vale o custo?

# Decisões Arquiteturais

## IA para desenvolvimento

Atualmente não será escolhida uma única IA.

Durante a especialização serão comparadas:

- ChatGPT
- Claude
- Gemini
- Claude Code
- GitHub Copilot
- Cursor
- Windsurf

A decisão final será baseada em produtividade, qualidade do código, documentação e custo-benefício.
