# Module 00 — Orientation & Engineering Environment

> "Build the environment once. Learn forever."

---

# Objective

This module prepares the entire engineering environment used throughout the AI Engineering Accelerator.

After completing this module, the student will have a complete professional workspace for learning, documenting, building, and publishing AI projects.

---

# Learning Goals

By the end of this module, you will be able to:

- Understand the specialization structure.
- Configure the GitHub organization.
- Configure the Knowledge Base repository.
- Configure NotebookLM.
- Configure XMind.
- Configure GitHub Projects (Kanban).
- Understand the documentation workflow.
- Understand the publication workflow.
- Understand how every module connects.
